1. 모든 요소를 선택하여 box-sizing: border-box; 로 크기 계산에 padding과 border를 포함하고, margin: 0; 으로 기본 바깥 여백을 제거합니다.

2. body 요소를 flex 컨테이너로 만들고, 세로·가로 중앙 정렬합니다.
배경색은 #f6f7fb, 높이는 100vh, 기본 글자색은 #111827 로 지정합니다.

3. .wrap 요소의 가로 너비를 420px 

4. .card 요소에 흰색 배경과 연한 테두리를 주고, 그림자(box-shadow)로 떠 있는 효과를 만듭니다.
모서리는 border-radius: 16px 로 둥글게 하고, 넘치는 내용은 overflow: hidden 으로 잘라냅니다.

5. .card-header 에 안쪽 여백(padding: 22px 22px 16px;)을 주고, 배경색은 #0ea5e9, 글자색은 흰색으로 설정합니다.

6. 카드 헤더 안의 h1 글자 크기를 20px 로 지정합니다.

7. .card-body 내부 여백을 위 20px, 좌우 22px, 아래 22px 로 지정합니다.

8. .field 요소 아래에 14px 간격을 주어 입력 항목 사이 여백을 만듭니다.

9. .input 입력창을 부모 너비 100%로 채우고, 안쪽 여백과 둥근 테두리를 적용합니다.
기본 포커스 테두리는 outline: none 으로 제거합니다.

10. 입력창이 포커스되면 테두리 색을 파란색으로 변경하고, 바깥쪽에 그림자 효과로 포커스 강조를 줍니다.
.input:focus {
  border-color: rgba(37, 99, 235, 0.6);
  box-shadow: 0 0 0 4px rgba(37, 99, 235, 0.12);
}

11. .row 를 flex 컨테이너로 만들어 내부 요소를 가로 배치하고, 양쪽 끝 정렬합니다.
위쪽은 6px, 아래쪽은 14px 여백을 줍니다.


12. .check 텍스트 색상을 회색( #6b7280)으로 지정합니다.

13. .link 글자 크기를 13px로 하고, 파란색 글자색(#2563eb)을 적용하며 밑줄을 제거합니다.


14. .btn 버튼을 가로 100%로 만들고, 파란 배경과 흰 글자색을 적용합니다.
둥근 모서리, 굵은 글씨, 마우스 포인터 효과를 설정합니다.


15. 버튼에 마우스를 올리면 투명도를 0.8로 낮춰 호버 효과를 줍니다.

16. .card-footer 에 위쪽 구분선을 넣고, 연한 배경색(#fafafa)과 중앙 정렬된 안내 문구 스타일을 지정합니다.
